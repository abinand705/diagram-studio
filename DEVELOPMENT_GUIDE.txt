2. SETUP INSTRUCTIONS
================================================================================

A. BACKEND SETUP
================

1. Create backend folder:
   mkdir backend
   cd backend

2. Initialize npm project:
   npm init -y

3. Install required packages:
   npm install express cors dotenv

4. Create .env file:
   OPENAI_API_KEY=your_key_here
   PORT=5000

5. Copy server.js to backend folder

6. Run backend:
   node server.js
   
   Expected output: 🚀 Smart Diagram Generator running on port 5000


B. FRONTEND SETUP
=================

1. Create React app (from project root):
   npx create-react-app frontend
   cd frontend

2. Install diagram library:
   npm install @xyflow/react

3. Copy App.js to frontend/src/

4. Run frontend:
   npm start
   
   Expected: Browser opens http://localhost:3000

4. API ENDPOINTS
================================================================================

BASE URL: http://localhost:5000

GET / 
  Test endpoint
  Response: { message: "API running" }

POST /generate-er
  Generate ER diagram from text
  Body: { text: "description" }
  Response: { entities: [], relationships: [] }

POST /generate-dfd
  Generate DFD from text
  Body: { text: "description" }
  Response: { processes: [], data_flows: [] }

GET /templates
  Get all available templates
  Response: { templates: [] }

GET /templates/:id
  Get specific template (e.g., "student_mgmt")
  Response: { entities: [], relationships: [] }

POST /save-diagram
  Save diagram to backend
  Body: { name, nodes, edges, type }
  Response: { success: true, diagramId }

GET /diagrams
  Get all saved diagrams
  Response: { total, diagrams: [] }

GET /diagrams/:id
  Get specific diagram
  Response: { id, name, nodes, edges, type }

DELETE /diagrams/:id
  Delete diagram
  Response: { success: true }

POST /validate
  Validate ER design
  Body: { entities, relationships }
  Response: { isValid, issues, suggestions }

POST /export/json
  Export diagram as JSON
  Body: { nodes, edges, name }
  Response: { data }

5. USAGE GUIDE
================================================================================

A. CREATING DIAGRAM WITH DRAG-AND-DROP
======================================

1. Choose diagram type (ER or DFD) from sidebar
2. Drag components from left sidebar to canvas
3. Connect components by dragging from handles
4. Customize node properties by clicking them
5. Add labels to relationships/flows
6. Save or export when done

B. USING TEMPLATES
==================

1. Click template name in sidebar
2. Diagram auto-loads with sample data
3. Modify entities/relationships as needed

C. GENERATING FROM TEXT
=======================

1. Type description in top input
2. Click "Generate" button
3. System creates diagram automatically
4. Edit and refine as needed

D. EXPORTING DIAGRAMS
=====================

PNG Export:
- Click "📥 PNG" button
- Downloads as image file

JSON Export:
- Click "📦 JSON" button
- Downloads as editable JSON

Save to Server:
- Click "💾 Save" button
- Diagram stored in backend

9. TROUBLESHOOTING
================================================================================

COMMON ISSUES:

1. "Failed to fetch" error
   - Make sure backend is running: node server.js
   - Check if port 5000 is available
   - Verify CORS is enabled

2. Nodes not dragging
   - Make sure you're holding from the node itself
   - Check that React Flow ref is properly set

3. Export not working
   - Ensure ReactFlow ref is connected
   - Check browser console for errors
   - Verify file permissions

4. Templates not loading
   - Verify backend is running
   - Check /templates endpoint
   - Inspect network tab in DevTools

15. CODE EXAMPLES
================================================================================

FETCHING ALL DIAGRAMS:
```javascript
const fetchDiagrams = async () => {
  const res = await fetch("http://localhost:5000/diagrams");
  const data = await res.json();
  console.log(data.diagrams);
};
```

CREATING CUSTOM NODE:
```javascript
const CustomNode = ({ data }) => (
  <div style={{ border: "2px solid blue", padding: "10px" }}>
    {data.label}
    <Handle type="target" position={Position.Top} />
    <Handle type="source" position={Position.Bottom} />
  </div>
);
```

VALIDATING DIAGRAM:
```javascript
const validate = async (entities, relationships) => {
  const res = await fetch("http://localhost:5000/validate", {
    method: "POST",
    body: JSON.stringify({ entities, relationships })
  });
  const result = await res.json();
  console.log(result.issues, result.suggestions);
};
```
