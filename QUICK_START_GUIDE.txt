================================================================================
SMART DIAGRAM GENERATOR - QUICK START GUIDE
================================================================================

⚡ GET YOUR DIAGRAM GENERATOR RUNNING IN 10 MINUTES

================================================================================
STEP 1: PROJECT SETUP (2 minutes)
================================================================================

1. Create project folder on Desktop:
   mkdir smart-diagram && cd smart-diagram

2. Open in VS Code:
   code .

3. Open terminal in VS Code (Terminal → New Terminal)

================================================================================
STEP 2: BACKEND SETUP (3 minutes)
================================================================================

1. Create backend folder:
   mkdir backend
   cd backend

2. Initialize project:
   npm init -y

3. Install packages:
   npm install express cors dotenv

4. Create .env file:
   - Right click in left panel → New File → .env
   - Add: OPENAI_API_KEY=sk-your-key-here
   - (You can skip AI and use mock mode)

5. Create server.js:
   - Right click in backend folder → New File → server.js
   - Copy the complete server.js code from outputs
   - Save (Ctrl + S)

6. Run backend:
   node server.js
   
   ✓ You should see: "🚀 Smart Diagram Generator running on port 5000"

7. Open new terminal (click +):
   Keep backend running in Terminal 1
   Use Terminal 2 for frontend

================================================================================
STEP 3: FRONTEND SETUP (4 minutes)
================================================================================

1. In Terminal 2, go to project root:
   cd ..

2. Create React app:
   npx create-react-app frontend
   
   ⏳ This takes 2-3 minutes...

3. Navigate to frontend:
   cd frontend

4. Install React Flow:
   npm install @xyflow/react

5. Replace App.js:
   - Open frontend/src/App.js
   - Delete all content
   - Paste the complete App.js code from outputs
   - Save (Ctrl + S)

6. Run frontend:
   npm start
   
   ✓ Browser should automatically open http://localhost:3000

================================================================================
STEP 4: TEST YOUR APP (1 minute)
================================================================================

1. In the browser (should be open):
   - Left sidebar: Choose "ER" or "DFD"
   - Drag "Entity" or "Process" from sidebar to canvas
   - Try dragging another component
   - Try connecting them with drag handles

2. Test AI generation:
   - Type in top input: "Student enrolls in course"
   - Click "🤖 Generate" button
   - Watch diagram appear!

3. Test export:
   - Click "📥 PNG" to download diagram as image
   - Click "📦 JSON" to download as JSON
   - Click "💾 Save" to save to backend

✓ SUCCESS! You have a working diagram generator!

================================================================================
COMMON CUSTOMIZATIONS
================================================================================

CHANGE COLORS:

Entity (ER) - In App.js, find EntityNode:
  border: "3px solid #1e40af",  ← Change this to your color

Process (DFD) - Find ProcessNode:
  background: "#fbbf24",        ← Change this to your color

ADD NEW SHAPE:

1. Create component in App.js:
   ```javascript
   const MyCustomNode = ({ data }) => (
     <div style={{ ... }}>{data.label}</div>
   );
   ```

2. Add to nodeTypes:
   ```javascript
   const nodeTypes = {
     ...existing,
     mycustom: MyCustomNode
   };
   ```

3. Add to sidebar in sidebarItems array

CHANGE DIAGRAM TYPE:

Modify sidebarItems array to show different components for each diagram type

================================================================================
TROUBLESHOOTING
================================================================================

"Failed to fetch" error:
→ Make sure backend is running in Terminal 1
→ Check if you see "🚀 running on port 5000"

Components not dragging:
→ Make sure you're dragging from the component item, not the label
→ Check browser console (F12) for errors

Export not working:
→ Ensure you have at least one component on canvas
→ Check if ReactFlow ref is connected properly

Blank canvas:
→ Make sure frontend is running (npm start)
→ Try refreshing browser
→ Check browser console for errors

Backend won't start:
→ Is port 5000 already in use?
  - Windows: netstat -ano | findstr :5000
  - Mac/Linux: lsof -i :5000
→ Try a different port in server.js: app.listen(5001)

Templates not showing:
→ Backend must be running
→ Check that GET /templates works: visit http://localhost:5000/templates

SUPPORT COMMANDS
